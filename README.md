Overview
--------

Simple boilerplate code for application, which will be based on [AngularJs](http://www.angularjs.org/) and [RequireJs](http://requirejs.org/)

Warning!
Works only via http!

[NodeJs](http://nodejs.org/) should be installed for proper work of build

Execute:

`npm install -g grunt-cli bower`

`npm install`

`bower install`

`grunt`

To build code (concatenate and minify), run: `grunt build`. Compiled version of application will be availabe at `/out` folder. Serve this folder via http, to access it
